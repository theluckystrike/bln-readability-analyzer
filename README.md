# bln-readability-analyzer

Free Text Readability Analyzer — Flesch-Kincaid Grade Level, Reading Ease, passive voice detection.

## Features

- Free to use, no login required
- Privacy-first: all processing happens in-browser
- Mobile-friendly responsive design
- Part of the [BeLikeNative](https://belikenative.com) open-source toolkit
